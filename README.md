# 📊 Sales & Customer Analytics Dashboard (Excel)

## 📌 Project Overview

This project presents an **interactive Sales & Customer Analytics Dashboard** built using Microsoft Excel. The dashboard provides insights into **sales performance, customer behavior, profitability, and operational efficiency** using advanced Excel techniques.

The primary objective of this project is to transform raw transactional data into meaningful business insights that can support **data-driven decision making**.

---

## 🎯 Objectives

* Analyze overall **sales and profit trends**
* Perform **customer segmentation using RFM (Recency, Frequency, Monetary) analysis**
* Identify **customer gain and loss (churn analysis)**
* Evaluate **shipping performance and delivery efficiency**
* Discover **regional and category-level insights**
* Build an **interactive dashboard with KPIs and slicers**

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Slicers & Interactive Filters
* RFM Analysis
* GETPIVOTDATA for dynamic KPIs
* Conditional Formatting
* Data Cleaning & Transformation

---

## 📂 Dataset

* Dataset used: **Sample Superstore Dataset**
* Contains:

  * Orders data (Sales, Profit, Quantity)
  * Customer details
  * Shipping information
  * Product categories and regions

---

## 📊 Key KPIs

* 💰 Total Sales
* 📈 Total Profit
* 🚚 Average Shipping Days
* 👥 Gained Customers
* ⚠️ Lost Customers
* 📦 Sales by Category
* 🌍 Sales by Region

---

## 📈 Dashboard Features

* Dynamic filtering using **Slicers (Year, Region, Category, Ship Mode)**
* Customer segmentation using **RFM analysis**
* Visualizations:

  * Sales trend over time
  * Customer distribution (pie chart)
  * Monthly sales performance
  * Shipping mode analysis
* KPI cards for quick business overview

---

## 🧠 Key Insights

* Sales show a **consistent upward trend from 2014 to 2017**
* Customer base is large, but **loyal customers are relatively fewer**
* **Customer churn is higher than acquisition**, indicating retention issues
* **Standard Class shipping** is most preferred by customers
* Sales peak during certain months, indicating **seasonal demand patterns**
* Profit is increasing, but not at the same rate as sales → **margin optimization needed**

---

## ⚠️ Business Challenges Identified

* High number of **lost customers compared to gained customers**
* Need to improve **customer retention strategies**
* Opportunity to increase **profit margins**

---

## ✅ Conclusion

The dashboard highlights that while the business is experiencing strong sales growth, **customer retention remains a critical issue**. Addressing churn and focusing on loyal customer segments can significantly improve long-term profitability and sustainability.
